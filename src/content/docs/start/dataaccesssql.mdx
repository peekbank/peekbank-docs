- **Hostname**: `34.210.173.143`
- **Port**: `3306`
- **Adapter**: MariaDB
- **Database**: `2025.1` (or any [supported release](../../peekbank/releases) like `2022.1`, `2021.1`)
- **Username**: `reader`
- **Password**: `gazeofraccoons`

You can then access the DB in your programming language of choice:
